# Sprint Tactics

Мини-игра в жанре пошаговой стратегии/тактики на тему разработки ПО.

## Запуск

1. Откройте файл `index.html` в браузере.
2. Игра запускается сразу, без установки зависимостей.

Или через локальный сервер:

1. Убедитесь, что установлен Node.js 18+.
2. Выполните `npm start`.
3. Откройте `http://localhost:10000`.

## Деплой на Render (Web Service)

Причина ошибки из логов: Render запускал `yarn start`, но в проекте не было `package.json`.

Теперь проект готов для `Web Service`:

1. `Runtime`: `Node`
2. `Build Command`: оставить пустым (или `npm install`)
3. `Start Command`: `npm start`

Render автоматически передаст переменную `PORT`, и сервер поднимется на нужном порту.

## Деплой на Render (Static Site)

Для этого проекта можно сделать еще проще:

1. Создайте сервис типа `Static Site`.
2. `Build Command`: пусто.
3. `Publish Directory`: `.`

Для чистого HTML/CSS/JS вариант `Static Site` обычно самый удобный.

## Правила

- Ваша команда: **Техлид**, **Frontend**, **QA инженер**.
- Противники: **Баг-рой**, **Legacy код**, **Дедлайн**.
- На каждом ходу каждый союзник может выполнить одно действие:
  - переместиться в доступную клетку;
  - атаковать врага в радиусе действия.
- После действий всех союзников начинается ход противника.
- Победа: уничтожить все угрозы.
- Поражение: потерять всю команду разработки.

## Управление

- Клик по союзнику: выбрать юнита.
- Подсветка синим: клетки для перемещения.
- Подсветка красным: враги в радиусе атаки.
- Кнопка `Завершить ход`: передать ход угрозам досрочно.
- Кнопка `Начать заново`: перезапуск матча.
